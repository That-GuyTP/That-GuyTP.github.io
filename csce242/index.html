CSCE 242

Description

CSCE 242 is a class about Web Development that teaches how to code in HTML, CSS, and JavaScript. This class also details useful examples of reworld situations where these languages could be best utilized.

Assignments ↓

Assignments ↑

Assignment 1 - Basic HTML

This assignment was a basic introduction to HTML coding as saw us creating a index.html with a basic topic that includes images and a table.

Assignment 2 - Basic CSS

This assignment added on to the first assignment but with a different index.html file/topic and the inclusion of a css stylesheet.

Assignment 3 - Page Layout

This assignment was a test of our knowledge on the "display:flex" css tag. We were tasked with recreating a website and having it work with 2 different aspect ratios.

Assignment 4 - Recreate CSS Page

We were tasked with recreating the USC's Graduate Admissions page.

Assignment 5 - JavaScript Buttons, Functions, and More

This assignment was a demo of three JavaScript features being a number that increases by 1 everytime you click it, an image that refreshes the page when clicked, and a slider that changes the x-position of an image when slid.

Assignment 6 - JavaScript Conditionals

This assignment further practiced JavaScript functions by having us create a dropdown navigation that only appears on smaller resolutions, a slider that changes the red value for a section's background, and an image that changes sizes based off the button clicked.

Assignment 7 - JavaScript Loop

In this assignment I was asked to create a input/button that took in a value and printed out a star based off a random left & top value inside a section. The star is then supposed to have a number associated to it when clicked.

Assignment 8 - JavaScript Array

An example of printing out a table using an array with images that can be interacted with once clicked.

Assignment 9 - JavaScript Classes

Using JavaScript classes I created a html page that shows a list of birds. When the birds are clicked a modal pops up and displays more information about the birds.

Assignment 10 - Json Parsing

In this assignment I used json parsing to display a list of ice cream flavors and their images. When you hover over the image, the flavor appears.
